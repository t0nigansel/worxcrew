# Design Reference

`CV_template.pdf` is a compact one-page A4 CV with:

- a pale blue-grey left sidebar
- a white main content area
- muted blue section headings and separator rules
- a photo in the upper-right header
- dense, small typography with tight spacing
- sidebar sections for technologies, certifications, and education
- main sections for profile, project excerpt, and work experience
- right-aligned project dates and small rectangular badges

The default editable assets in this folder mirror that layout:

- `cv_reference.css`
- `cv_preview_reference.html.j2`
- `cv_pdf_reference.tex.j2`

The previous showcase layout remains available as the legacy modern layout.
