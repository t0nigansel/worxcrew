# Interviewvorbereitung - Senior QA / Test Engineer

## Frage 1

**Wahrscheinliche Frage:** Wie wuerden Sie Ihren Beitrag in einer Rolle als Senior QA / Test Engineer beschreiben?

**Warum wahrscheinlich:** Die Ausschreibung legt Gewicht auf Security, AI, R.

**Starke Antwort:** Fokussieren Sie auf konkrete Wirkung, zum Beispiel auf Ergebnisse aus TI-Portal, Elektronische Patientenakte (EPA), und verbinden Sie diese mit messbarer Verbesserung in Qualitaet, Sicherheit oder Automatisierung.

## Frage 2

**Wahrscheinliche Frage:** Wie gehen Sie mit komplexen Qualitaets- oder Security-Risiken im Team um?

**Warum wahrscheinlich:** Ihre Profilstaerken und die Variante default deuten auf einen beratenden, strukturierenden Beitrag hin.

**Starke Antwort:** Beschreiben Sie einen klaren Ablauf aus Risikoanalyse, Priorisierung, Stakeholder-Abstimmung und Umsetzung. Verweisen Sie auf belastbare Referenzen wie TI-Portal, Elektronische Patientenakte (EPA).

## Frage 3

**Wahrscheinliche Frage:** Wie halten Sie Ihr Wissen aktuell und wie uebertragen Sie es in die Praxis?

**Warum wahrscheinlich:** Aktuelle Weiterbildungen wie Red Teaming LLM Applications, Applied ChatGPT for Cybersecurity machen diesen Punkt plausibel.

**Starke Antwort:** Verbinden Sie Weiterbildung mit einem konkreten Transfer in Projekte, Prozesse oder Standards, statt nur Zertifikate aufzuzaehlen.
