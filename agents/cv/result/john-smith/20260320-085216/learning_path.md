# Learning Path

## Naechster sinnvoller Schritt

Der naechste logische Schritt ist eine Position in Richtung **Principal Java Developer**. Die staerksten Hebel dafuer liegen in Software Architecture, Distributed Systems, Technical Leadership, Modernization.

## Wie Sie dorthin kommen

Bauen Sie gezielt sichtbare Referenzen auf, die fachliche Tiefe und Wirkung kombinieren: ein belastbares Projektbeispiel, ein klarer fachlicher Schwerpunkt und eine sichtbare Rolle in Standards, Coaching oder Automatisierung.

## Hilfreiche Zertifizierungen

Priorisieren Sie Zertifizierungen oder Lernpfade wie AWS Certified Developer, Oracle Java SE 17 Developer, Spring Professional, Spring Professional Certification. Wichtig ist, dass jede Weiterbildung direkt in Projektpraxis oder Positionierung ueberfuehrt wird.
